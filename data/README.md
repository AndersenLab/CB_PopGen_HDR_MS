# Data
This directory contains external data retrieved from other publications, CaeNDR, or from independent analysis carried out with other computational pipelines.

## Directory Structure

data/
- briggsae_genome_files/
- briggsae_isotypes/
- elegans_isotypes/
- Marie_anne_2013_and_Richard_2011_paper/
- Thomas_2015_tree_annotations/

## briggsae_genome_files

Contains QX1410 genome dictionary and chromosomal domain breakpoints. Retrieved from Stevens et al. 2021.

## briggsae_isotypes
Isotype assignments for every _C. briggsae_ strain. Output of https://github.com/AndersenLab/isotype-nf. Also available in https://caendr.org/data/data-release/.

## elegans_isotypes
Isotype assignments for every _C. elegans_ strain. Output of https://github.com/AndersenLab/isotype-nf. Also available in https://caendr.org/data/data-release/.
